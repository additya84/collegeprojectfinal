BCA Syllabus

Himachal Pradesh University (HPU)

Semester 1 Subjects & Marks Distribution

Detailed Syllabus - Semester 1

BCA101 - Mathematics-I

Unit 1: Sets, Relations, Functions, Types of Functions, Composition of Functions, Inverse Functions.

Unit 2: Matrices, Types of Matrices, Matrix Operations, Determinants, Inverse of Matrix, Cramer's Rule.

Unit 3: Limits, Continuity, Differentiation, Rules of Differentiation, Maxima and Minima.

Unit 4: Integration, Methods of Integration, Definite and Indefinite Integrals, Applications of Integration.

Unit 5: Differential Equations, Formation of Differential Equations, Solution of First Order Differential Equations.

BCA102 - Applied English

Unit 1: Grammar, Parts of Speech, Articles, Tenses, Voice, Narration.

Unit 2: Vocabulary Building, Synonyms, Antonyms, One Word Substitution, Idioms and Phrases.

Unit 3: Reading Comprehension, Passage Reading, Answer Writing Skills.

Unit 4: Letter Writing, Formal Letter, Informal Letter, Job Application Letter.

Unit 5: Report Writing, Communication Skills, Presentation Skills, Group Discussion.

BCA103 - Computer Fundamentals

Unit 1: Introduction to Computers, Characteristics of Computers, Generations of Computers, Types of Computers.

Unit 2: Input Devices, Output Devices, Memory Units, Storage Devices.

Unit 3: Number System, Binary, Decimal, Octal and Hexadecimal Number Systems, Conversions.

Unit 4: Operating System, Functions of OS, Types of Operating Systems, DOS and Windows.

Unit 5: Internet, WWW, Web Browser, Search Engine, Email, Computer Security.

BCA104 - C Programming

Unit 1: Introduction to C Language, Features of C, Structure of C Program, Character Set, Tokens, Keywords, Identifiers, Variables, Constants, Data Types.

Unit 2: Operators and Expressions, Arithmetic Operators, Relational Operators, Logical Operators, Assignment Operators, Increment and Decrement Operators, Type Conversion.

Unit 3: Decision Making and Looping Statements, if, if-else, Nested if, switch, for loop, while loop, do-while loop, break and continue statements.

Unit 4: Arrays, Strings, One Dimensional Array, Two Dimensional Array, String Functions, User Defined Functions, Recursion.

Unit 5: Pointers, Pointer Arithmetic, Structures, Union, File Handling, fopen(), fclose(), fread(), fwrite().

BCA105 - Office Automation Tools

Unit 1: Introduction to MS Office, Features of MS Office, Word Processing Concepts.

Unit 2: MS Word, Formatting, Tables, Mail Merge, Templates, Page Setup.

Unit 3: MS Excel, Worksheets, Formulas, Functions, Charts, Data Sorting and Filtering.

Unit 4: MS PowerPoint, Slides, Themes, Animation, Transition Effects, Presentation Creation.

Unit 5: Internet Applications, Web Browsing, Search Engines, Email Services, Cloud Storage.
