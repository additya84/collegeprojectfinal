Analogy

Analogy is one of the easiest and most frequently asked topics in Placement Tests, SSC, Banking and Railway exams. In analogy questions, candidates identify relationships between words, numbers, letters or figures and apply the same relationship to another pair.

This chapter includes:

  • Concept Explanation
  • Shortcut Tricks
  • 10 Solved Questions
  • Recommended Videos
  • PDF Resources

Types of Analogy Questions

1. Word Analogy

Doctor : Hospital :: Teacher : School

2. Number Analogy

5 : 25 :: 7 : 49

3. Alphabet Analogy

A : Z :: B : Y

4. Meaning Based Analogy

Hot : Cold :: Day : Night

5. Cause and Effect Analogy

Study : Success :: Exercise : Fitness

Shortcut Tricks

1. Identify relationship first.

2. Check whether relation is based on function, place or characteristic.

3. In number analogy check square, cube and prime patterns first.

4. In alphabet analogy check position values.

5. Always compare both terms before checking options.

Solved Questions

Q1. Doctor : Hospital :: Teacher : ?

Doctor works in Hospital.
Teacher works in School.

Answer = School

Q2. 8 : 64 :: 9 : ?

8² = 64
9² = 81

Answer = 81

Q3. Book : Reading :: Food : ?

Book is used for Reading.
Food is used for Eating.

Answer = Eating

Q4. Bird : Nest :: Lion : ?

Bird lives in Nest.
Lion lives in Den.

Answer = Den

Q5. 4 : 16 :: 11 : ?

4² = 16
11² = 121

Answer = 121

Q6. Pen : Write :: Knife : ?

Pen is used to write.
Knife is used to cut.

Answer = Cut

Q7. Cat : Kitten :: Dog : ?

Kitten is young one of Cat.
Puppy is young one of Dog.

Answer = Puppy

Q8. 6 : 36 :: 13 : ?

6² = 36
13² = 169

Answer = 169

Q9. Fish : Water :: Human : ?

Fish lives in Water.
Human lives on Land.

Answer = Land

Q10. A : Z :: C : ?

A ↔ Z
B ↔ Y
C ↔ X

Answer = X
