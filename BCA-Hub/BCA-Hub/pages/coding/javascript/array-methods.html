📦 JavaScript Arrays

Array Methods

Learn important JavaScript Array Methods with examples and outputs.

What is an Array?

An array is used to store multiple values in a single variable. 

let fruits = ["Apple", "Banana", "Mango"];

push()

Adds an element at the end of the array.

let fruits = ["Apple", "Banana"];

fruits.push("Mango");

console.log(fruits);

["Apple", "Banana", "Mango"]

pop()

Removes the last element from the array.

let fruits = ["Apple", "Banana", "Mango"];

fruits.pop();

console.log(fruits);

["Apple", "Banana"]

shift()

Removes the first element from the array.

let fruits = ["Apple", "Banana", "Mango"];

fruits.shift();

console.log(fruits);

["Banana", "Mango"]

unshift()

Adds an element at the beginning of the array.

let fruits = ["Banana", "Mango"];

fruits.unshift("Apple");

console.log(fruits);

["Apple", "Banana", "Mango"]

length

Returns the number of elements in an array.

let fruits = ["Apple", "Banana", "Mango"];

console.log(fruits.length);

3

includes()

Checks whether an element exists in the array.

let fruits = ["Apple", "Banana", "Mango"];

console.log(fruits.includes("Mango"));

true

indexOf()

Returns the index position of an element.

let fruits = ["Apple", "Banana", "Mango"];

console.log(fruits.indexOf("Banana"));

1

join()

Joins all array elements into a string.

let fruits = ["Apple", "Banana", "Mango"];

console.log(fruits.join(" - "));

Apple - Banana - Mango

reverse()

Reverses the array elements.

let fruits = ["Apple", "Banana", "Mango"];

fruits.reverse();

console.log(fruits);

["Mango", "Banana", "Apple"]

sort()

Sorts array elements alphabetically.

let fruits = ["Mango", "Apple", "Banana"];

fruits.sort();

console.log(fruits);

["Apple", "Banana", "Mango"]
