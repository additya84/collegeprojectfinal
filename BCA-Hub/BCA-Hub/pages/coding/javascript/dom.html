🌐 JavaScript DOM

DOM Manipulation

Learn Document Object Model (DOM) in JavaScript including selecting elements, changing content and handling events.

What is DOM?

DOM stands for Document Object Model. It represents the HTML document as objects which JavaScript can access and modify.

DOM allows JavaScript to change HTML content, styles and attributes dynamically.

getElementById()

Used to select an element using its id. 

Hello World

querySelector()

Selects the first matching element. 

let heading = document.querySelector("h1");
console.log(heading);

querySelectorAll()

Selects all matching elements. 

let items = document.querySelectorAll("p");

console.log(items);
