📘 Introduction to C

C is a general-purpose, procedural programming language developed by Dennis Ritchie in 1972 at Bell Laboratories.

It is fast, efficient and widely used in operating systems, embedded systems, compilers and system software.

Important: C is often called the "Mother of Programming Languages" because many languages like C++, Java and C# are influenced by it.

🏗 Structure of a C Program

#include <stdio.h>

int main()
{
    printf("Hello World");

    return 0;
}

Explanation

  • #include <stdio.h> → Header file for input/output functions.
  • int main() → Program execution starts here.
  • printf() → Displays output on screen.
  • return 0; → Indicates successful execution.

📦 Variables

Variables are memory locations used to store data values.

Syntax

datatype variable_name;

Example 1

int age = 19;

printf("%d", age);
Output: 19

Here int is datatype, age is variable name and 19 is stored value.

Example 2

float marks = 89.5;

printf("%.1f", marks);
Output: 89.5

🔢 Data Types

Data Type Description Example
int Stores integer values 10
float Stores decimal values 10.5
char Stores characters 'A'
double Stores large decimal values 10.55555

⭐ Important Points

  • C is case sensitive.
  • Every statement ends with semicolon (;).
  • Keywords cannot be used as variable names.
  • Variable names cannot start with numbers.
  • main() function is mandatory in every C program.
