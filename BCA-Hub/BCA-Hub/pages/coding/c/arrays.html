Arrays in C

Learn Arrays in C Programming with theory, syntax, examples, programs and outputs.

📘 Introduction to Arrays

An array is a collection of similar data items stored in contiguous memory locations. Arrays are used to store multiple values of the same data type using a single variable name.

Example: Instead of creating 100 variables for storing marks, we can create a single array of size 100.

📝 Declaration of Array

datatype array_name[size];

int marks[5];

Explanation

  • int → Data type
  • marks → Array name
  • 5 → Number of elements

⚡ Initialization of Array

int numbers[5] = {10,20,30,40,50};
Array Elements: 10 20 30 40 50

🔍 Accessing Array Elements

#include <stdio.h>

int main()
{
    int arr[5] = {10,20,30,40,50};

    printf("%d", arr[2]);

    return 0;
}
Output: 30

Array indexing starts from 0. Therefore arr[2] refers to the third element.

⌨️ Input and Output in Arrays

#include <stdio.h>

int main()
{
    int arr[5];

    for(int i=0;i<5;i++)
    {
        scanf("%d",&arr[i]);
    }

    for(int i=0;i<5;i++)
    {
        printf("%d ",arr[i]);
    }

    return 0;
}
Input: 10 20 30 40 50

Output: 10 20 30 40 50

➕ Program 1 : Sum of Array Elements

#include <stdio.h>

int main()
{
    int arr[5]={10,20,30,40,50};
    int sum=0;

    for(int i=0;i<5;i++)
    {
        sum = sum + arr[i];
    }

    printf("Sum = %d",sum);

    return 0;
}
Output: Sum = 150

🏆 Program 2 : Largest Element

#include <stdio.h>

int main()
{
    int arr[5]={10,90,40,20,60};
    int max=arr[0];

    for(int i=1;i<5;i++)
    {
        if(arr[i]>max)
        {
            max=arr[i];
        }
    }

    printf("Largest = %d",max);

    return 0;
}
Output: Largest = 90

🔄 Program 3 : Reverse Array

#include <stdio.h>

int main()
{
    int arr[5]={10,20,30,40,50};

    for(int i=4;i>=0;i--)
    {
        printf("%d ",arr[i]);
    }

    return 0;
}
Output: 50 40 30 20 10

⭐ Important Points

  • Arrays store elements of same data type.
  • Array indexing starts from 0.
  • Arrays use contiguous memory locations.
  • Array size cannot be changed after declaration.
  • Arrays provide fast access using indexes.
