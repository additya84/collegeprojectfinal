Strings in C

Learn Strings in C Programming including declaration, initialization, string functions and important programs.

📘 Introduction to Strings

A string is a collection of characters stored inside an array and terminated with a null character '\0'.

Example: "Hello" is a string whereas 'A' is a character.

📝 Declaration of String

char name[20];

Here name can store up to 19 characters and one null character.

⚡ Initialization of String

char name[] = "Additya";
Output: Additya

⌨️ Input and Output of Strings

#include

int main()
{
    char name[20];

    scanf("%s",name);

    printf("%s",name);

    return 0;
}
Input: Additya

Output: Additya

📏 strlen() Function

strlen() returns the length of a string. 

#include
#include

int main()
{
    char str[]="BCA Hub";

    printf("%d",strlen(str));

    return 0;
}
Output: 7

📋 strcpy() Function

strcpy() copies one string into another string. 

#include
#include

int main()
{
    char str1[20];
    char str2[]="Hello";

    strcpy(str1,str2);

    printf("%s",str1);

    return 0;
}
Output: Hello

➕ strcat() Function

strcat() joins two strings. 

#include
#include

int main()
{
    char str1[30]="BCA ";
    char str2[]="Hub";

    strcat(str1,str2);

    printf("%s",str1);

    return 0;
}
Output: BCA Hub

⚖️ strcmp() Function

strcmp() compares two strings. 

#include
#include

int main()
{
    char str1[]="Hello";
    char str2[]="Hello";

    printf("%d",strcmp(str1,str2));

    return 0;
}
Output: 0

If output is 0, strings are equal.

🔄 Program 1 : Reverse a String

#include

int main()
