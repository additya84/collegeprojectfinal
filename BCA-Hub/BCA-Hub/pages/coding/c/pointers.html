Pointers in C

Learn pointers, memory addresses, dereferencing and pointer arithmetic with examples, outputs and explanations.

📘 Introduction to Pointers

A pointer is a variable that stores the memory address of another variable.

Instead of storing a value directly, pointers store the address where the value is stored.

📝 Syntax of Pointer

datatype *pointer_name;

int *ptr;
float *ptr;
char *ptr;

📍 Address Operator (&)

The & operator is used to get the address of a variable. 

#include

int main()
{
    int num = 10;

    printf("%p",&num);

    return 0;
}
Output: 0x61ff08

The output may be different on every computer because memory addresses change.

🔍 Dereference Operator (*)

The * operator is used to access the value stored at the memory address. 

#include

int main()
{
    int x = 50;

    int *ptr = &x;

    printf("%d",*ptr);

    return 0;
}
Output: 50

✏️ Changing Value using Pointer

#include

int main()
{
    int num = 10;

    int *ptr = #

    *ptr = 25;

    printf("%d",num);

    return 0;
}
Output: 25

➕ Pointer Arithmetic

We can perform arithmetic operations on pointers such as increment and decrement. 

#include

int main()
{
    int arr[3]={10,20,30};

    int *ptr = arr;

    printf("%d\n",*ptr);

    ptr++;

    printf("%d",*ptr);

    return 0;
}

10
20

📊 Pointers and Arrays

#include

int main()
{
    int arr[5]={10,20,30,40,50};

    int *ptr = arr;

    for(int i=0;i<5;i++)
    {
        printf("%d ",*(ptr+i));
    }

    return 0;
}
Output: 10 20 30 40 50

❌ Null Pointer

int *ptr = NULL;

A null pointer does not point to any memory location.
