🔄 C Programming

Loops in C

Learn For Loop, While Loop, Do While Loop and Nested Loops with examples, outputs and explanations.

📘 Introduction to Loops

Loops are used to execute a block of code repeatedly until a condition becomes false. They reduce code repetition and make programs shorter and easier to manage.

Example: Printing numbers from 1 to 100 using 100 print statements is difficult, so we use loops.

🔁 For Loop

The for loop is used when the number of iterations is known. 

for(initialization; condition; increment/decrement)
{
    // code
}

Example

#include

int main()
{
    for(int i=1;i<=5;i++)
    {
        printf("%d ",i);
    }

    return 0;
}
Output: 1 2 3 4 5

🔄 While Loop

The while loop checks the condition first and then executes the code block. 

while(condition)
{
    // code
}

#include

int main()
{
    int i=1;

    while(i<=5)
    {
        printf("%d ",i);
        i++;
    }

    return 0;
}
Output: 1 2 3 4 5

♻️ Do While Loop

The do while loop executes the code at least once, even if the condition is false. 

do
{
    // code
}
while(condition);

#include

int main()
{
    int i=1;

    do
    {
        printf("%d ",i);
        i++;
    }
    while(i<=5);

    return 0;
}
Output: 1 2 3 4 5

🧩 Nested Loops

A loop inside another loop is called a nested loop. 

#include

int main()
{
    for(int i=1;i<=3;i++)
    {
        for(int j=1;j<=3;j++)
        {
            printf("* ");
        }

        printf("\n");
    }

    return 0;
}

💻 Program 1: Print Table of 5

#include

int main()
{
    for(int i=1;i<=10;i++)
    {
        printf("5 x %d = %d\n",i,5*i);
    }

    return 0;
}

5 x 1 = 5
5 x 2 = 10
5 x 3 = 15
...
5 x 10 = 50

💻 Program 2: Sum of First 10 Numbers

#include

int main()
{
    int sum=0;

    for(int i=1;i<=10;i++)
    {
        sum += i;
    }

    printf("Sum = %d",sum);

    return 0;
}
Output: Sum = 55

💻 Program 3: Factorial of Number

#include

int main()
{
    int n=5,fact=1;

    for(int i=1;i<=n;i++)
    {
        fact*=i;
    }

    printf("Factorial = %d",fact);

    return 0;
}
Output: Factorial = 120

  • while loop checks condition before execution.
  • do while loop executes at least once.
  • Nested loops are mainly used for patterns and matrices.
