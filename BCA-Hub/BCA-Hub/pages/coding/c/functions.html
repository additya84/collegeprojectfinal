⚙️ C Programming

Functions in C

Learn Functions in C Programming with syntax, examples, return values and recursion.

📘 Introduction to Functions

A function is a block of code that performs a specific task. Functions help us avoid writing the same code repeatedly.

Example: Instead of writing addition code 10 times, we can create an add() function and call it whenever needed.

📝 Syntax of Function

return_type function_name(parameters)
{
    // code
}

Example:

int add(int a, int b)
{
    return a+b;
}

📚 Types of Functions

  • Library Functions
  • User Defined Functions

Library Functions: Functions already available in C such as printf(), scanf(), sqrt().

User Defined Functions: Functions created by the programmer.

📌 Function Declaration

int add(int, int);

Function declaration tells the compiler about the function name, return type and parameters.

⚡ Function Definition

int add(int a, int b)
{
    return a+b;
}

📞 Function Calling

result = add(10,20);

Function calling transfers control from main() to the function.

💻 Program 1: Addition using Function

#include

int add(int a,int b)
{
    return a+b;
}

int main()
{
    int result;

    result = add(10,20);

    printf("%d",result);

    return 0;
}
Output: 30
  • add() function receives two numbers.
  • return sends the result back to main().
  • main() prints the returned value.

💻 Program 2: Square of Number

#include

int square(int n)
{
    return n*n;
}

int main()
{
    printf("%d",square(5));

    return 0;
}
Output: 25

🚀 Function with No Arguments and No Return Value

#include

void greet()
{
    printf("Welcome to BCA Hub");
}

int main()
{
    greet();

    return 0;
}
Output: Welcome to BCA Hub

📥 Function with Arguments but No Return Value

#include

void display(int age)
{
    printf("%d",age);
}

int main()
{
    display(19);

    return 0;
}
Output: 19

📤 Function with Arguments and Return Value

#include

int multiply(int a,int b)
{
    return a*b;
}

int main()
{
    printf("%d",multiply(10,5));

    return 0;
}
Output: 50

🔁 Recursive Function

A recursive function is a function that calls itself. 

#include

int fact(int n)
{
    if(n==1)
        return 1;

    return n*fact(n-1);
}

int main()
{
    printf("%d",fact(5));

    return 0;
}
Output: 120
