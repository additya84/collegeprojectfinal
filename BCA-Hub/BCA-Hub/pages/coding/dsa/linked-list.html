🔗 Data Structure

Linked List

Learn Linked List including types, insertion, deletion and traversal operations with examples.

📘 What is a Linked List?

A Linked List is a linear data structure where elements are stored in nodes and each node points to the next node.

Each node contains two parts: Data + Address of Next Node

🧩 Structure of Node

+------+-------+
| Data | Next  |
+------+-------+

The data field stores the value and next stores the address of the next node.

📚 Types of Linked List

  • Singly Linked List
  • Doubly Linked List
  • Circular Linked List
  • Circular Doubly Linked List

1️⃣ Singly Linked List

10 → 20 → 30 → 40 → NULL

Each node points to the next node only.

2️⃣ Doubly Linked List

NULL ← 10 ⇄ 20 ⇄ 30 ⇄ 40 → NULL

Each node contains previous and next pointers.

3️⃣ Circular Linked List

10 → 20 → 30 → 40
↑             ↓
└─────────────┘

The last node points back to the first node.

🚶 Traversal Operation

Traversal means visiting every node in the linked list. 

10 → 20 → 30 → 40 → NULL

➕ Insertion Operation

Before:
10 → 20 → 40 → NULL

Insert 30

After:
10 → 20 → 30 → 40 → NULL

➖ Deletion Operation

Before:
10 → 20 → 30 → 40 → NULL

Delete 30

After:
10 → 20 → 40 → NULL

💻 Example Program

struct Node
{
    int data;
    struct Node *next;
};

This is the basic structure of a linked list node in C.

✅ Advantages of Linked List

  • Dynamic size.
  • Easy insertion and deletion.
  • No memory wastage.
  • Efficient memory utilization.

❌ Disadvantages of Linked List

  • Extra memory required for pointers.
  • No direct access using index.
  • Traversal is slower than arrays.
