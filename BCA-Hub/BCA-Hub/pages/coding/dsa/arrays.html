📦 Data Structure

Arrays

Learn Array concepts including Traversal, Insertion, Deletion and Searching with examples and programs.

📘 What is an Array?

An Array is a collection of elements of the same data type stored in contiguous memory locations.

Example: Instead of creating 100 variables, we can store 100 values in a single array.

📝 Declaration of Array

int arr[5];

This creates an integer array capable of storing 5 elements.

⚡ Initialization of Array

int arr[5] = {10,20,30,40,50};
Output: 10 20 30 40 50

🔍 Traversal Operation

Traversal means visiting each element of an array one by one. 

for(int i=0;i<5;i++)
{
    printf("%d ",arr[i]);
}
Output: 10 20 30 40 50

➕ Insertion Operation

Insertion means adding a new element at a specific position. 

Array Before:
10 20 40 50

Insert 30 at position 3

Array After:
10 20 30 40 50

➖ Deletion Operation

Deletion means removing an element from an array. 

Array Before:
10 20 30 40 50

Delete 30

Array After:
10 20 40 50

🔎 Linear Search

int arr[5]={10,20,30,40,50};
int key=30;

for(int i=0;i<5;i++)
{
    if(arr[i]==key)
    {
        printf("Element Found");
    }
}
Output: Element Found

💻 Program Example

#include

int main()
{
    int arr[5]={10,20,30,40,50};

    for(int i=0;i<5;i++)
    {
        printf("%d ",arr[i]);
    }

    return 0;
}
Output: 10 20 30 40 50

✅ Advantages of Arrays

  • Easy to access elements using index.
  • Stores multiple values using one variable.
  • Fast access to data.
  • Efficient memory usage.

❌ Disadvantages of Arrays

  • Fixed size.
  • Insertion and deletion are costly.
  • Stores only same type of elements.
