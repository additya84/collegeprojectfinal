📚 Data Structure

Stack

Learn Stack Data Structure including Push, Pop, Peek operations and Stack applications.

📘 What is Stack?

Stack is a linear data structure that follows the LIFO (Last In First Out) principle.

The element inserted last is removed first.

📚 Stack Representation

| 40 | ← Top
| 30 |
| 20 |
| 10 |
-----

Insertion and deletion both take place from the top of the stack.

➕ Push Operation

Push operation inserts an element into the stack. 

Before:

| 30 |
| 20 |
| 10 |

Push 40

After:

| 40 |
| 30 |
| 20 |
| 10 |

➖ Pop Operation

Pop operation removes the top element from the stack. 

Before:

| 40 |
| 30 |
| 20 |
| 10 |

Pop

After:

| 30 |
| 20 |
| 10 |

👀 Peek Operation

Peek operation returns the top element without removing it. 

Top Element = 40

💻 Stack Implementation in C

#define SIZE 5

int stack[SIZE];
int top = -1;

🌍 Applications of Stack

  • Function Calls
  • Undo and Redo Operations
  • Expression Evaluation
  • Recursion
  • Browser History

✅ Advantages of Stack

  • Simple implementation.
  • Efficient insertion and deletion.
  • Useful for recursion and backtracking.

❌ Disadvantages of Stack

  • No random access to elements.
  • Limited size in array implementation.
  • Searching is slower.
