🌳 Data Structure

Tree

Learn Tree Data Structure including Binary Tree, Binary Search Tree and Tree Traversal Algorithms.

What is a Tree?

Tree is a non-linear hierarchical data structure consisting of nodes connected by edges.

A tree contains one Root Node and multiple Child Nodes.

Tree Representation

          A
        /   \
       B     C
      / \   / \
     D   E F   G

Terminologies of Tree

  • Root Node
  • Parent Node
  • Child Node
  • Leaf Node
  • Sibling Node
  • Sub Tree
  • Height of Tree
  • Degree of Node

Binary Tree

A Binary Tree is a tree in which each node can have at most two children. 

        A
       / \
      B   C
     / \
    D   E

Binary Search Tree (BST)

A Binary Search Tree is a binary tree where left child is smaller and right child is greater than the parent node. 

        50
       /  \
     30    70
    / \   / \
   20 40 60 80

Preorder Traversal

Visit Root → Left → Right 

Tree:

      A
     / \
    B   C

Traversal:
A B C

Inorder Traversal

Visit Left → Root → Right 

Traversal:
B A C

Postorder Traversal

Visit Left → Right → Root 

Traversal:
B C A

Applications of Tree

  • File System Structure
  • Database Indexing
  • Compiler Design
  • Routing Tables
  • Decision Making Systems

Advantages of Tree

  • Efficient Searching.
  • Hierarchical Data Representation.
  • Faster insertion and deletion in BST.

Disadvantages of Tree

  • Complex implementation.
  • Requires extra memory.
  • Traversal algorithms are complex.
