🕸️ Data Structure

Graph

Learn Graph Data Structure including vertices, edges, graph representation and traversal algorithms.

📘 What is a Graph?

A Graph is a non-linear data structure consisting of vertices (nodes) and edges connecting those vertices.

Graph = Vertices + Edges

🔹 Components of Graph

  • Vertex (Node)
  • Edge
  • Degree
  • Path
  • Cycle

📍 Vertex

A vertex is a point in a graph representing an object or entity. 

Vertices = {A, B, C, D}

🔗 Edge

An edge connects two vertices in a graph. 

Edges = {(A,B), (B,C), (C,D)}

📚 Types of Graph

  • Directed Graph
  • Undirected Graph
  • Weighted Graph
  • Unweighted Graph
  • Cyclic Graph
  • Acyclic Graph

🧾 Adjacency Matrix Representation

    A B C
A   0 1 1
B   1 0 1
C   1 1 0

In an adjacency matrix, 1 means an edge exists and 0 means no edge exists.

📋 Adjacency List Representation

A -> B -> C
B -> A -> C
C -> A -> B

🚶 Breadth First Search (BFS)

BFS visits nodes level by level using a Queue. 

Start Node: A

Traversal:
A B C D E

🌲 Depth First Search (DFS)

DFS visits nodes as deep as possible before backtracking. 

Start Node: A

Traversal:
A B D E C

✅ Advantages of Graph

  • Represents complex relationships easily.
  • Used in maps and navigation systems.
  • Useful in social networks.
  • Efficient for network representation.

🌍 Applications of Graph

  • Google Maps
  • Social Media Networks
  • Computer Networks
  • Recommendation Systems
  • Routing Algorithms
