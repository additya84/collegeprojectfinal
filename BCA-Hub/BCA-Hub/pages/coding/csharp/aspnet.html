📘 What is ASP.NET?

ASP.NET is a web development framework developed by Microsoft for building dynamic websites, web applications and APIs.

ASP.NET uses C# as its primary programming language and runs on the .NET platform.

⭐ Features of ASP.NET

  • Fast and High Performance
  • Cross Platform Support
  • Secure Framework
  • Easy Database Connectivity
  • Supports MVC Architecture
  • Developed and Maintained by Microsoft

🏛 MVC Architecture

ASP.NET mainly follows the MVC architecture pattern.

  • Model → Handles application data and business logic.
  • View → Responsible for the user interface.
  • Controller → Handles user requests and responses.
MVC helps separate business logic from the user interface, making applications easier to manage and maintain.

🎮 Controller

A Controller handles incoming requests and returns responses. 

using Microsoft.AspNetCore.Mvc;

public class HomeController : Controller
{
    public IActionResult Index()
    {
        return View();
    }
}

📦 Model

A Model represents application data and business rules. 

public class Student
{
    public int Id { get; set; }

    public string Name { get; set; }

    public int Age { get; set; }
}

🖥 View

Views are responsible for displaying data to users. 

<h1>Welcome to BCA Hub</h1>

<p>This page is rendered using ASP.NET MVC View.</p>

🛣 Routing

Routing determines which controller and action method should handle an incoming request. 

app.MapControllerRoute(
    name: "default",
    pattern: "{controller=Home}/{action=Index}/{id?}"
);
