📚 C# Collections

Collections in C#

Learn C# Collections including List, Dictionary, Queue, Stack and HashSet with examples and outputs.

📘 What are Collections?

Collections are classes used to store and manage groups of objects dynamically. Unlike arrays, collections can grow or shrink automatically.

Collections belong to the System.Collections.Generic namespace.

📋 List<T>

List is a dynamic array that allows adding and removing elements easily. 

using System;
using System.Collections.Generic;

List numbers = new List();

numbers.Add(10);
numbers.Add(20);
numbers.Add(30);

foreach(int num in numbers)
{
    Console.WriteLine(num);
}

10
20
30

📚 Stack<T>

Stack follows the LIFO (Last In First Out) principle. 

using System;
using System.Collections.Generic;

Stack stack = new Stack();

stack.Push(10);
stack.Push(20);
stack.Push(30);

Console.WriteLine(stack.Pop());
Output: 30

🚶 Queue<T>

Queue follows the FIFO (First In First Out) principle. 

using System;
using System.Collections.Generic;

Queue queue = new Queue();

queue.Enqueue(10);
queue.Enqueue(20);
queue.Enqueue(30);

Console.WriteLine(queue.Dequeue());
Output: 10

🗂 Dictionary<TKey, TValue>

Dictionary stores data in key-value pairs. 

using System;
using System.Collections.Generic;

Dictionary students =
new Dictionary();

students.Add(101,"Rahul");
students.Add(102,"Aman");

Console.WriteLine(students[101]);
Output: Rahul

🎯 HashSet<T>

HashSet stores unique values only and automatically removes duplicates. 

using System;
using System.Collections.Generic;

HashSet numbers =
new HashSet();

numbers.Add(10);
numbers.Add(20);
numbers.Add(20);
numbers.Add(30);

foreach(int num in numbers)
{
    Console.WriteLine(num);
}

10
20
30
