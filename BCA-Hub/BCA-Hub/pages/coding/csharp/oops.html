🏛️ Object Oriented Programming

OOP Concepts in C#

Learn Object Oriented Programming concepts in C# including Classes, Objects, Encapsulation, Inheritance, Polymorphism and Abstraction.

📘 What is OOP?

Object Oriented Programming (OOP) is a programming paradigm based on objects and classes. It helps in creating reusable, secure and maintainable applications.

C# is a fully Object Oriented Programming language.

🏗️ Class

A class is a blueprint or template used to create objects. 

class Student
{
    public int RollNo;
    public string Name;
}

📦 Object

An object is an instance of a class. 

Student s1 = new Student();
Here s1 is an object of Student class.

💻 Example of Class and Object

using System;

class Student
{
    public string Name;

    public void Display()
    {
        Console.WriteLine(Name);
    }
}

class Program
{
    static void Main()
    {
        Student s1 = new Student();

        s1.Name = "Additya";

        s1.Display();
    }
}
Output: Additya

🔒 Encapsulation

Wrapping data and methods into a single unit is called Encapsulation. 

class BankAccount
{
    private int balance;

    public void SetBalance(int amount)
    {
        balance = amount;
    }

    public int GetBalance()
    {
        return balance;
    }
}

👨‍👩‍👦 Inheritance

Inheritance allows one class to acquire the properties and methods of another class. 

class Animal
{
    public void Eat()
    {
        Console.WriteLine("Eating...");
    }
}

class Dog : Animal
{
}

🔄 Polymorphism

Polymorphism means one method can perform multiple tasks. 

class Print
{
    public void Show(int x)
    {
        Console.WriteLine(x);
    }

    public void Show(string y)
    {
        Console.WriteLine(y);
    }
}

🎭 Abstraction

Abstraction means hiding implementation details and showing only essential features. 

abstract class Shape
{
    public abstract void Draw();
}

🔑 Access Modifiers

Modifier Description
public Accessible everywhere
private Accessible only inside class
protected Accessible inside derived classes
internal Accessible inside same assembly

✅ Advantages of OOP

  • Code Reusability
  • Easy Maintenance
  • Better Security
  • Modular Programming
  • Easy Debugging
