📘 Introduction to C#

C# is an object-oriented programming language developed by Microsoft. It is widely used for desktop applications, web applications, APIs and game development using Unity.

C# is pronounced as "C Sharp".

🏗 Structure of a C# Program

using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("Hello World");
    }
}
Output: Hello World
