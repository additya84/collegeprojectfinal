📚 C++ STL

Standard Template Library (STL)

Learn C++ STL including Vector, Stack, Queue, Set, Map and Pair with examples and outputs.

📘 What is STL?

STL stands for Standard Template Library. It is a collection of ready-made classes and functions that helps programmers write efficient code quickly.

STL mainly contains Containers, Iterators and Algorithms.

🧩 Components of STL

  • Containers
  • Algorithms
  • Iterators

📦 Vector

Vector is a dynamic array that can grow or shrink in size automatically. 

#include <iostream>
#include <vector>
using namespace std;

int main()
{
    vector v = {10,20,30,40};

    for(int x : v)
    {
        cout << x << " ";
    }

    return 0;
}
Output: 10 20 30 40

📚 Stack

Stack follows LIFO (Last In First Out) principle. 

#include <iostream>
#include <stack>
using namespace std;

int main()
{
    stack s;

    s.push(10);
    s.push(20);
    s.push(30);

    cout << s.top();

    return 0;
}
Output: 30

🚶 Queue

Queue follows FIFO (First In First Out) principle. 

#include <iostream>
#include <queue>
using namespace std;

int main()
{
    queue q;

    q.push(10);
    q.push(20);
    q.push(30);

    cout << q.front();

    return 0;
}
Output: 10

🎯 Set

Set stores unique elements in sorted order. 

#include <iostream>
#include <set>
using namespace std;

int main()
{
    set s = {50,20,10,20,30};

    for(int x : s)
    {
        cout << x << " ";
    }

    return 0;
}
Output: 10 20 30 50

🗺️ Map

Map stores data in key-value pairs. 

#include <iostream>
#include <map>
using namespace std;

int main()
{
    map student;

    student[101] = "Rahul";
    student[102] = "Aman";

    cout << student[101];

    return 0;
}
Output: Rahul

👥 Pair

Pair stores two values together. 

#include <iostream>
#include <utility>
using namespace std;

int main()
{
    pair p;

    p.first = 101;
    p.second = "Additya";

    cout << p.first << " " << p.second;

    return 0;
}
Output: 101 Additya

🔁 Iterators

Iterators are used to traverse elements in STL containers. 

vector::iterator it;

for(it=v.begin(); it!=v.end(); it++)
{
    cout << *it << " ";
}

  • Easy implementation of complex tasks.
