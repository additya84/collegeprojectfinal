🏛️ Object Oriented Programming

OOP Concepts in C++

Learn Object Oriented Programming concepts in C++ including Classes, Objects, Encapsulation, Abstraction, Inheritance and Polymorphism with examples and programs.

📘 What is OOP?

Object Oriented Programming (OOP) is a programming approach based on objects and classes. It helps in writing reusable, secure and maintainable code.

C++ is one of the most popular Object Oriented Programming languages.

🏗️ Class

A class is a blueprint or template for creating objects. 

class Student
{
    public:
        int roll;
        string name;
};

📦 Object

An object is an instance of a class. 

Student s1;

Here s1 is an object of Student class.

💻 Example of Class and Object

#include <iostream>
using namespace std;

class Student
{
public:
    string name;
    int roll;

    void display()
    {
        cout << "Name: " << name << endl;
        cout << "Roll No: " << roll;
    }
};

int main()
{
    Student s1;

    s1.name = "Additya";
    s1.roll = 101;

    s1.display();

    return 0;
}

Name: Additya
Roll No: 101

🔐 Access Specifiers

Access specifiers control the accessibility of class members.

  • Public → Accessible everywhere.
  • Private → Accessible only inside the class.
  • Protected → Accessible inside class and derived class.

📦 Encapsulation

Wrapping data and functions into a single unit is called Encapsulation. 

class Bank
{
private:
    int balance;

public:
    void setBalance(int b)
    {
        balance = b;
    }

    int getBalance()
    {
        return balance;
    }
};

🎭 Abstraction

Hiding unnecessary implementation details and showing only important information to the user is called Abstraction.

Example: ATM machine shows only options like Withdraw and Deposit but hides internal processing.

👨‍👩‍👧 Inheritance

Inheritance allows one class to acquire properties and methods of another class. 

class Animal
{
};

class Dog : public Animal
{
};

🔄 Polymorphism

Polymorphism means one thing having many forms. 

class Print
{
public:
    void show(int x)
    {
        cout << x;
    }

    void show(string y)
    {
        cout << y;
    }
};

✅ Advantages of OOP

  • Code Reusability
  • Better Security
  • Easy Maintenance
  • Improved Productivity
  • Modular Programming
