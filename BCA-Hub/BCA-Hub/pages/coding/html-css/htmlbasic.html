📄 HTML Fundamentals

HTML Basics

Learn HTML basics including tags, headings, paragraphs, links, images, lists, tables and forms.

What is HTML?

HTML stands for HyperText Markup Language. It is used to create the structure of web pages.

HTML is the backbone of every website.

Basic Structure of HTML

<!DOCTYPE html>
<html>
<head>
    <title>My Website</title>
</head>

<body>
    Hello World
</body>
</html>

Heading Tags

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

Paragraph Tag

<p>
This is a paragraph.
</p>

Link Tag

<a href="https://google.com">
Google
</a>

Image Tag

<img src="image.jpg" alt="Image">

Lists

<ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
</ul>

Table Tag

<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>

    <tr>
        <td>Additya</td>
        <td>19</td>
    </tr>
</table>

Form Tag

<form>
    <input type="text"
    placeholder="Enter Name">

    <button>
        Submit
    </button>
</form>

Important HTML Tags

  • <html>
  • <head>
  • <body>
  • <h1> to <h6>
  • <p>
  • <a>
  • <img>
  • <table>
  • <form>
  • <div>

Advantages of HTML

  • Easy to learn.
  • Platform independent.
  • Supported by all browsers.
  • Used for creating web pages.
