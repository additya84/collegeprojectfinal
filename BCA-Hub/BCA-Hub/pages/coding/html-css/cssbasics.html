CSS Basics

Learn CSS basics including selectors, colors, fonts, borders, margins, padding and layouts.

What is CSS?

CSS stands for Cascading Style Sheets. It is used to style and design HTML web pages.

HTML creates the structure and CSS makes it beautiful.

CSS Syntax

selector{
    property: value;
}
Example: h1{ color: blue; }

Inline CSS

<h1 style="color:red;">
Hello World
</h1>

Internal CSS

<style>
h1{
    color:blue;
}
</style>

External CSS

<link rel="stylesheet"
href="style.css">

Selectors

  • Element Selector
  • Class Selector
  • ID Selector
  • Universal Selector
h1{
    color:red;
}

.title{
    color:blue;
}

#heading{
    color:green;
}

Colors

h1{
    color:red;
}

body{
    background-color:lightblue;
}

Fonts

p{
    font-size:20px;
    font-family:Arial;
    font-weight:bold;
}

Border

div{
    border:2px solid black;
}

Margin and Padding

div{
    margin:20px;
    padding:20px;
}

Width and Height

div{
    width:300px;
    height:150px;
}

Display Property

div{
    display:flex;
}
