CSS Box Model

Learn CSS Box Model including Content, Padding, Border and Margin with examples.

What is CSS Box Model?

Every HTML element is treated as a rectangular box in CSS. This box consists of Content, Padding, Border and Margin.

Box Model = Content + Padding + Border + Margin

Box Model Structure

+-----------------------+
|        Margin         |
|  +-----------------+  |
|  |     Border      |  |
|  | +-------------+ |  |
|  | |   Padding   | |  |
|  | | +---------+ | |  |
|  | | | Content | | |  |
|  | | +---------+ | |  |
|  | +-------------+ |  |
|  +-----------------+  |
+-----------------------+

Content

Content is the actual text, image or element inside the box. 

div{
    width:300px;
    height:150px;
}

Padding

Padding creates space between content and border. 

div{
    padding:20px;
}

Border

Border surrounds the padding and content. 

div{
    border:2px solid black;
}

Margin

Margin creates space outside the border. 

div{
    margin:20px;
}

Complete Example

div{
    width:300px;
    padding:20px;
    border:2px solid black;
    margin:30px;
}

Example Output

Margin
 └── Border
      └── Padding
           └── Content

  • Controls spacing effectively.
  • Makes responsive design easier.
  • Improves UI structure.
