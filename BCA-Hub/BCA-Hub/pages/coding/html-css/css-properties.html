🎨 CSS Styling

CSS Properties

Learn important CSS properties used for styling websites, layouts and responsive web pages.

Color Property

The color property changes the text color. 

h1{
    color:red;
}

Background Color

The background-color property changes the background color. 

body{
    background-color:lightblue;
}

Font Size

The font-size property changes the size of text. 

p{
    font-size:20px;
}

Font Family

The font-family property changes the font style. 

body{
    font-family:Arial;
}

Margin

Margin creates space outside an element. 

div{
    margin:20px;
}

Padding

Padding creates space inside an element. 

div{
    padding:20px;
}

Border

Border adds a line around an element. 

div{
    border:2px solid black;
}

Width and Height

Width and height set the size of an element. 

div{
    width:300px;
    height:150px;
}

Display Property

The display property controls how elements appear. 

div{
    display:flex;
}

Position Property

Position controls element placement on the page. 

div{
    position:relative;
}

Flexbox Property

.container{
    display:flex;
    justify-content:center;
    align-items:center;
}

Grid Property

.container{
    display:grid;
    grid-template-columns:1fr 1fr 1fr;
}

Transition Property

button{
    transition:0.3s;
}

Transform Property

.card:hover{
    transform:scale(1.1);
}
