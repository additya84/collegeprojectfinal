📱 Responsive Web Design

Responsive Design

Learn Responsive Web Design using Media Queries, Flexible Layouts and Mobile Friendly Techniques.

What is Responsive Design?

Responsive Design is a web design approach that makes websites look good on all devices such as mobiles, tablets, laptops and desktops.

One website can adapt automatically to different screen sizes.

Why Responsive Design?

  • Better user experience.
  • Works on all devices.
  • Improves website accessibility.
  • Improves SEO ranking.

Viewport Meta Tag

<meta name="viewport"
content="width=device-width, initial-scale=1.0">

This tag helps the website scale properly on mobile devices.

Media Query

Media Queries apply different styles for different screen sizes. 

@media(max-width:768px)
{
    body{
        background:lightblue;
    }
}

Responsive Grid Example

.container{
    display:grid;
    grid-template-columns:
    repeat(auto-fit,minmax(250px,1fr));
}

Responsive Flexbox Example

.container{
    display:flex;
    flex-wrap:wrap;
    gap:20px;
}

Responsive Images

img{
    max-width:100%;
    height:auto;
}

Responsive Typography

h1{
    font-size:clamp(2rem,5vw,4rem);
}

Example Media Query

@media(max-width:600px)
{
    .hero h1{
        font-size:32px;
    }

    .container{
        padding:15px;
    }
}

Best Practices

  • Use flexible layouts.
  • Use percentage widths.
  • Use media queries.
  • Optimize images for mobile devices.
  • Test on multiple screen sizes.
