CSS Flexbox

Learn CSS Flexbox including flex container, flex items and important flex properties.

What is Flexbox?

Flexbox is a CSS layout model used to arrange items in rows or columns efficiently and responsively.

Flexbox makes alignment and spacing easier.

Basic Syntax

.container{
    display:flex;
}

flex-direction

Defines the direction of items. 

.container{
    display:flex;
    flex-direction:row;
}

flex-direction:column;

justify-content

Aligns items horizontally. 

.container{
    justify-content:center;
}
  • flex-start
  • center
  • flex-end
  • space-between
  • space-around
  • space-evenly

align-items

Aligns items vertically. 

.container{
    align-items:center;
}
  • stretch
  • center
  • flex-start
  • flex-end

gap Property

Adds spacing between flex items. 

.container{
    gap:20px;
}

flex-wrap

Allows items to move to the next line. 

.container{
    flex-wrap:wrap;
}

  • Responsive layouts.
  • Less code required.
  • Better control over spacing.
