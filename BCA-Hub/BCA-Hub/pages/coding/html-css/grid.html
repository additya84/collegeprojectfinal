CSS Grid

Learn CSS Grid including Grid Container, Grid Items and important Grid properties with examples.

What is CSS Grid?

CSS Grid is a two-dimensional layout system used to arrange elements in rows and columns.

Grid makes creating complex layouts simple and responsive.

Basic Syntax

.container{
    display:grid;
}

grid-template-columns

Defines the number and size of columns. 

.container{
    display:grid;
    grid-template-columns:
    1fr 1fr 1fr;
}

grid-template-rows

Defines the number and size of rows. 

.container{
    grid-template-rows:
    100px 100px;
}

gap Property

Adds spacing between rows and columns. 

.container{
    gap:20px;
}

grid-column

Specifies how many columns an item should span. 

.item1{
    grid-column:1/3;
}

grid-row

Specifies how many rows an item should span. 

.item1{
    grid-row:1/3;
}

Responsive Grid Example

.container{
    display:grid;
    grid-template-columns:
    repeat(auto-fit,minmax(250px,1fr));
}

Complete Example

.container{
    display:grid;
    grid-template-columns:
    repeat(3,1fr);
    gap:20px;
}

.box{
    padding:20px;
}

Advantages of CSS Grid

  • Easy row and column layouts.
  • Responsive design support.
  • Less code required.
  • Better control over positioning.
