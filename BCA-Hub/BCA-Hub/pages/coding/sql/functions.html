SQL Functions

Learn SQL Functions including COUNT, SUM, AVG, MAX and MIN with examples and outputs.

What are SQL Functions?

SQL Functions are built-in functions used to perform calculations and operations on database data.

SQL Functions help reduce code and improve query performance.

COUNT() Function

COUNT() returns the total number of rows. 

SELECT COUNT(*)
FROM Students;

5

SUM() Function

SUM() returns the total sum of a column. 

SELECT SUM(Marks)
FROM Students;

420

AVG() Function

AVG() returns the average value of a column. 

SELECT AVG(Marks)
FROM Students;

84

MAX() Function

MAX() returns the highest value in a column. 

SELECT MAX(Marks)
FROM Students;

95

MIN() Function

MIN() returns the smallest value in a column. 

SELECT MIN(Marks)
FROM Students;

65

UPPER() Function

UPPER() converts text into uppercase letters. 

SELECT UPPER(Name)
FROM Students;

RAHUL
AMAN
ROHIT

LOWER() Function

LOWER() converts text into lowercase letters. 

SELECT LOWER(Name)
FROM Students;

rahul
aman
rohit

ROUND() Function

ROUND() rounds a decimal value. 

SELECT ROUND(85.678,2);

85.68

Function Summary

Function Purpose
