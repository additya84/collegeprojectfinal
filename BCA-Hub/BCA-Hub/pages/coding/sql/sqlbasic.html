🗄️ Database Fundamentals

SQL Basics

Learn SQL basics including Databases, Tables, Data Types and basic SQL commands.

What is SQL?

SQL stands for Structured Query Language. It is used to store, retrieve and manage data in databases.

SQL is the standard language used for relational databases.

What is a Database?

A Database is an organized collection of data that can be easily accessed and managed. 

Examples:
• Student Database
• Employee Database
• Banking Database

What is a Table?

A Table stores data in rows and columns. 

+----+---------+-----+
| ID | Name    | Age |
+----+---------+-----+
| 1  | Rahul   | 20  |
| 2  | Aman    | 21  |
+----+---------+-----+

Data Types

Data Type Description
INT Stores integer values
VARCHAR Stores text values
FLOAT Stores decimal values
DATE Stores date values

CREATE DATABASE

CREATE DATABASE CollegeDB;

CREATE TABLE

CREATE TABLE Students(
    Id INT,
    Name VARCHAR(50),
    Age INT
);

INSERT INTO

INSERT INTO Students
VALUES(1,'Rahul',20);

SELECT Statement

SELECT * FROM Students;

1 Rahul 20

Advantages of SQL

  • Easy to learn and use.
  • Fast data retrieval.
  • Supports large databases.
  • Used in almost every application.
