🔍 SQL Queries

SQL Queries

Learn SQL Queries including SELECT, WHERE, ORDER BY, GROUP BY and LIMIT with examples.

What is a Query?

A Query is a command used to retrieve or manipulate data from a database.

SQL Queries help us communicate with databases.

SELECT Query

SELECT is used to retrieve data from a table. 

SELECT * FROM Students;

Id   Name    Age
1    Rahul   20
2    Aman    21

WHERE Clause

WHERE is used to filter records based on a condition. 

SELECT * FROM Students
WHERE Age > 20;

2    Aman    21

ORDER BY Clause

ORDER BY is used to sort records in ascending or descending order. 

SELECT * FROM Students
ORDER BY Name ASC;

SELECT * FROM Students
ORDER BY Age DESC;

DISTINCT Query

DISTINCT removes duplicate values from the result. 

SELECT DISTINCT City
FROM Students;

LIMIT Clause

LIMIT is used to restrict the number of rows returned. 

SELECT * FROM Students
LIMIT 5;

GROUP BY Clause

GROUP BY groups rows with the same values. 

SELECT City, COUNT(*)
FROM Students
GROUP BY City;

HAVING Clause

HAVING is used with GROUP BY to filter grouped data. 

SELECT City, COUNT(*)
FROM Students
GROUP BY City
HAVING COUNT(*) > 1;

LIKE Operator

LIKE is used for pattern matching. 

SELECT * FROM Students
WHERE Name LIKE 'R%';

BETWEEN Operator

SELECT * FROM Students
WHERE Age BETWEEN 18 AND 25;
