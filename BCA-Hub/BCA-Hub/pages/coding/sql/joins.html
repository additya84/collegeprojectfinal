SQL Joins

Learn SQL Joins including INNER JOIN, LEFT JOIN, RIGHT JOIN and FULL JOIN with examples.

What is a Join?

SQL Join is used to combine rows from two or more tables based on a related column between them.

Joins help us retrieve data from multiple tables at the same time.

Example Tables

Students Table

ID   Name
1    Rahul
2    Aman
3    Rohit


Marks Table

ID   Marks
1    85
2    90
4    75

INNER JOIN

INNER JOIN returns only matching records from both tables. 

SELECT Students.Name, Marks.Marks
FROM Students
INNER JOIN Marks
ON Students.ID = Marks.ID;

Rahul   85
Aman    90

LEFT JOIN

LEFT JOIN returns all records from the left table and matching records from the right table. 

SELECT Students.Name, Marks.Marks
FROM Students
LEFT JOIN Marks
ON Students.ID = Marks.ID;

Rahul   85
Aman    90
Rohit   NULL

RIGHT JOIN

RIGHT JOIN returns all records from the right table and matching records from the left table. 

SELECT Students.Name, Marks.Marks
FROM Students
RIGHT JOIN Marks
ON Students.ID = Marks.ID;

Rahul   85
Aman    90
NULL    75

FULL JOIN

FULL JOIN returns all records when there is a match in either table. 

SELECT Students.Name, Marks.Marks
FROM Students
FULL OUTER JOIN Marks
ON Students.ID = Marks.ID;

Rahul   85
Aman    90
Rohit   NULL
NULL    75

Join Summary

Join Type Returns
INNER JOIN Matching records only
LEFT JOIN All left table records
RIGHT JOIN All right table records
FULL JOIN All records from both tables
