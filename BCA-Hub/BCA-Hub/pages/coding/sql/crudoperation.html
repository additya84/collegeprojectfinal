➕ SQL CRUD Operations

CRUD Operations

Learn Create, Read, Update and Delete operations in SQL with examples and outputs.

What is CRUD?

CRUD stands for Create, Read, Update and Delete. These are the basic operations performed on database records.

CRUD = Create + Read + Update + Delete

Create (INSERT)

INSERT command is used to add new records into a table. 

INSERT INTO Students
VALUES(1,'Rahul',20);

1 Rahul 20

Read (SELECT)

SELECT command is used to retrieve data from a table. 

SELECT * FROM Students;

Id   Name    Age
1    Rahul   20

Update (UPDATE)

UPDATE command is used to modify existing records. 

UPDATE Students
SET Age = 21
WHERE Id = 1;

Id   Name    Age
1    Rahul   21

Delete (DELETE)

DELETE command is used to remove records from a table. 

DELETE FROM Students
WHERE Id = 1;

Record Deleted Successfully

CRUD Summary Table

Operation Command Purpose
Create INSERT Add New Record
Read SELECT Retrieve Data
Update UPDATE Modify Data
Delete DELETE Remove Data

Advantages of CRUD Operations

  • Easy data management.
  • Efficient database operations.
  • Supports all database applications.
  • Improves data handling.
